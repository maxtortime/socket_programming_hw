### 프로그램 동작법
* 압축파일을 푼다.
* 쉘에 다음과 같이 입력한다.
    ```sh
    $ make
    $ ls
    client  client.c  makefile  server  server.c
    ```
* client 와 server가 각각 서버와 클라이언트의 실행파일이다.
